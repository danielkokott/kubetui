# kubetui

<!-- ## Features -->


## Command Option

```sh
kubetui 0.4.1
kosay <ekr59uv25@gmail.com>


USAGE:
    kubetui [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -s, --split-mode <direction>
            Window split mode [default: vertical]
            [possible values: vertical, v, horizontal, h]
```
